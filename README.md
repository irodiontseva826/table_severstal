# Интерактивная таблица

![alt text](image.png)

## Функциональность
- Раскрытие строк, имеющих дочерние элементы
- Фильтрация по свойству `isActive`
- Сортировка по полям `balance` и `email`

## Стек
- JS React
- Module CSS
- Vite

## Установка и запуск
```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```
